Tags h1 - h6: Os elementos HTML de cabeçalho da seção

Voltar ao topo

Os elementos HTML h1 - h6 representam seis níveis de título de seção. h1 é o nível de seção mais alto e h6 é o mais baixo.

Exemplo:

Elementos de cabeçalho são implementados em seis níveis, h1 é o mais importante e h6 é o de menor importância. Um elemento de cabeçalho descreve brevemente o tópico da seção em que ele está. As informações de cabeçalho podem ser usadas por agentes de usuário, por exemplo, para construir uma tabela de conteúdos para um documento automaticamente.

Tag p: O elemento Parágrafo

Voltar ao topo

O elemento HTML p representa um parágrafo. Em mídias visuais, parágrafos são representados como blocos indentados de texto com a primeira letra avançada e separados por linhas em branco. Já em HTML, parágrafos são usados para agrupar conteúdos relacionados de qualquer tipo, como imagens e campos de um formulário.

Tag mark:

Voltar ao topo

O Elemento HMTL mark representa um trecho de destaque em um texto , por exemplo, uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular. Por Exemplo, pode ser utilizado em uma página que mostra os resultados de uma busca onde todas as instâncias da palavra pesquisadas receberam destaque.

Tag small: o elemento de comentário lateral

Voltar ao topo

O elemento HTML small representa comentários laterais e letras pequenas, como direitos autorais e texto legal, independentemente de sua apresentação estilizada. Por padrão, ele renderiza o texto dentro dele um tamanho de fonte menor, como de pequeno para x-pequeno.

Tag i: O elemento Texto Idiomático

Voltar ao topo

O elemento HTML representa um intervalo de texto que se destaca do texto normal por algum motivo, como texto idiomático, termos técnicos, designações taxonômicas, entre outros. Historicamente, eles foram apresentados usando o tipo itálico, que é a fonte original da nomenclatura desse elemento.

Tag u: O elemento de anotação não articulada (sublinhado)

Voltar ao topo

O elemento HTML representa um intervalo de texto embutido que deve ser renderizado de uma forma que indique que possui uma anotação não textual. Isso é renderizado por padrão como um simples sublinhado sólido, mas pode ser alterado usando CSS.

Tag strong: O elemento de forte importância

Voltar ao topo

O elemento HTML indica que seu conteúdo tem forte importância, seriedade ou urgência. Os navegadores normalmente renderizam o conteúdo em negrito.

Tag li: O elemento Item de lista

Voltar ao topo

O elemento HTML é usado para representar um item em uma lista. Ele deve estar contido em um elemento pai: uma lista ordenada, uma lista não ordenada ou um menu. Em menus e listas não ordenadas, os itens de lista geralmente são exibidos usando marcadores. Em listas ordenadas, eles geralmente são exibidos com um contador crescente à esquerda, como um número ou uma letra.

Tag a: O elemento Âncora

Voltar ao topo

O elemento HTML (ou elemento âncora ), com seu atributo , cria um hiperlink para páginas da Web, arquivos, endereços de e-mail, locais na mesma página ou qualquer outra coisa que um URL possa endereçar. O conteúdo de cada um 'a' deve indicar o destino do link. Se o hrefatributo estiver presente, pressionar a tecla Enter enquanto estiver focado no 'a' elemento o ativará.

Tag hr: O elemento Intervalo Temático (Regra Horizontal)

Voltar ao topo

O elemento HTML representa uma quebra temática entre elementos de nível de parágrafo: por exemplo, uma mudança de cena em uma história ou uma mudança de tópico dentro de uma seção.

Tag blockquote: O elemento Block Quotation

Voltar ao topo

O elemento HTML indica que o texto incluído é uma citação estendida. Normalmente, isso é renderizado visualmente por recuo (consulte Notas para saber como alterá-lo). Uma URL para a fonte da citação pode ser fornecida usando o atributo, enquanto uma representação de texto da fonte pode ser fornecida usando o elemento.

Tag font: O Elemento para estilização de texto

Voltar ao topo

O elemento HTML define o tamanho da fonte, a cor e a face do seu conteúdo.

Notas:

Preterido: este recurso não é mais recomendado. Embora alguns navegadores ainda possam suportá-lo, ele pode já ter sido removido dos padrões da Web relevantes, pode estar em processo de eliminação ou pode ser mantido apenas para fins de compatibilidade. Evite usá-lo e atualize o código existente, se possível; consulte a tabela de compatibilidade na parte inferior desta página para orientar sua decisão. Esteja ciente de que esse recurso pode deixar de funcionar a qualquer momento.

Aviso: Não use este elemento. Use as propriedades CSS Fonts para estilizar o texto.

Tag del: O elemento de texto excluído

Voltar ao topo

O elemento HTML representa um intervalo de texto que foi excluído de um documento. Isso pode ser usado ao renderizar "mudanças de faixa" ou informações de diferenças de código-fonte, por exemplo. O elemento pode ser usado para o propósito oposto: para indicar o texto que foi adicionado ao documento.

Tag abbr: O elemento Abreviação

Voltar ao topo

O elemento HTML representa uma abreviação ou acrônimo.

Ao incluir uma abreviatura ou acrônimo, forneça uma expansão completa do termo em texto simples no primeiro uso, juntamente com o abbr símbolo para marcar a abreviatura. Isso informa ao usuário o que a abreviação ou sigla significa.

O atributo opcional titlepode fornecer uma expansão para a abreviação ou acrônimo quando uma expansão completa não estiver presente. Isso fornece uma dica para os agentes de usuário sobre como anunciar/exibir o conteúdo enquanto informa a todos os usuários o que a abreviação significa. Se presente, titledeve conter esta descrição completa e nada mais.
